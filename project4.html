Eventtracker XIV

Stay up to date with events from the game Final Fantasy XIV.

What is Eventtracker XIV?

As of 2024 the critically acclaimed MMORPG Final Fantasy XIV is one of the most popular games of its genre. Sadly it lacks many quality of life features for its players, one of which is an intuitive way to check current or upcoming events. While the official website delivers posts on such events, they remain hard to find and do not provide any visual representation outside an event banner which contains the start and end dates. I, as a long standing fan, offer a simple solution with this project.

Eventtracker XIV is a passion project, which displays relevant event information from the official website such as name, duration and theme in an easy to understand calendar format. Users have the ability to filter events by their type, display upcoming maintenance and read on for more information by following links back to the official website.

Project Overview

This application uses a community run api to retain data from Final Fantasies website The Loadstone and in addition parses certain data from The Loadstone directly. The backend has been written in Python, and fulfills 2 main functionalities. On one hand it filters the incoming data. Certain data, such as dates, is only available in the form of an image. Which is why Eventtracker XIV also utilizes the open source optical character recognition software Tesseract. On the other hand it saves to and retrieves data from a MySQL Database.

Eventtracker XIV runs as an app using the Flask framework. The frontend uses plain HTML and JS to create a dynamic website. My responsibilities during this project included, but were not limited to:

  • Designing the application
  • Finding solutions for and implementing data collection
  • Implementing Tesseract
  • Adding RegEx filters
  • Creating and implementing the database
  • Creating SQL commands in Python
  • Seting up Flask
  • Creating the HTML and CSS
  • Finding solutions for and implementing a dynamic calander in JS
  • Adding all remaining features
